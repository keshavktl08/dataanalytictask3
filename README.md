# Marketing Funnel & Customer Conversion Analysis

## Project Overview

This project analyzes marketing and lead funnel data to understand how users move through the stages **Visitors → Leads → Customers**.
Using **Python, Pandas, and Matplotlib in Jupyter Notebook**, the analysis explores customer behavior, marketing channels, and conversion rates using the **Bank Marketing Dataset**.

The goal of this project is to gain insights into how marketing efforts convert potential customers into actual customers.

---

## Objectives

* Analyze marketing funnel stages.
* Understand customer conversion behavior.
* Identify factors affecting customer subscription.
* Visualize insights using graphs and charts.

---

## Tools & Technologies

* Python
* Jupyter Notebook
* Pandas
* Matplotlib

---

## Dataset

The dataset used in this project is **bank-full.csv**, which contains marketing campaign data of a bank.

Key features in the dataset include:

* Age
* Job
* Marital Status
* Education
* Balance
* Contact Method
* Campaign Information
* Target Variable (`y`) indicating whether the client subscribed to the product.

---

## Steps Performed

### 1. Data Loading

The dataset was loaded using **Pandas**.

### 2. Data Exploration

* Checked dataset structure
* Identified important columns
* Explored summary statistics

### 3. Customer Conversion Analysis

Analyzed how many customers subscribed to the bank product.

### 4. Marketing Channel Analysis

Studied different contact methods used during marketing campaigns.

### 5. Data Visualization

Used **Matplotlib** to create visualizations such as:

* Bar charts
* Histograms
* Conversion analysis graphs

---

## Key Insights

* Majority of customers did not subscribe to the product.
* Certain job categories showed higher engagement.
* Marketing contact method influenced campaign success.
* Customer demographics play a role in conversion behavior.

---

## Conclusion

This project demonstrates how **data analysis can help businesses understand customer behavior and improve marketing strategies**.
By analyzing marketing funnel data, organizations can optimize campaigns and increase conversion rates.
